# Notle Run 개인정보 처리방침

**시행일**: 2026년 5월 21일
**최종 개정일**: 2026년 5월 21일

Notle Run("앱")은 사용자의 개인정보를 소중히 다루며, 「개인정보 보호법」(PIPA) 및 관련 법령을 준수합니다. 본 처리방침은 앱이 처리하는 개인정보의 항목·목적·보관·이용·제공·파기에 대한 사항을 규정합니다.

---

## 1. 처리하는 개인정보 항목

Notle Run은 러닝 기록과 통계를 제공하기 위해 다음 정보를 처리합니다.

### 1-1. 위치 정보
- **수집 항목**: GPS 위도·경도·고도·이동 속도·이동 경로
- **수집 시점**: 사용자가 러닝 세션을 시작한 때부터 종료할 때까지 (백그라운드 포함)
- **수집 방법**: 단말기의 GPS 모듈을 통해 직접 수집
- **선택성**: 위치 권한 거부 시 러닝 추적·영토 기능을 이용할 수 없습니다.

### 1-2. 건강 정보 (선택)
- **수집 항목**: 심박수, 걸음수, 이동 거리, 워크아웃 메타데이터
- **수집 시점**: 사용자가 Apple HealthKit 연동에 동의한 경우, 러닝 세션 중·후
- **수집 방법**: Apple HealthKit API를 통해 사용자의 명시적 동의 하에 읽기/쓰기
- **선택성**: 거부 시 심박수·HealthKit 동기화 기능만 비활성화되며, 다른 기능은 정상 동작합니다.

### 1-3. 모션 센서 정보
- **수집 항목**: 가속도, 케이던스(분당 걸음수)
- **수집 시점**: 러닝 세션 진행 중
- **수집 방법**: 단말기의 모션 센서를 통해 직접 수집

### 1-4. 자동 생성 정보
- **수집 항목**: 러닝 세션 ID, 시작·종료 시각, 영토 타일 좌표, 챌린지 진행도
- **수집 방법**: 앱 사용 중 자동 생성

---

## 2. 개인정보 처리 목적

| 항목 | 처리 목적 |
|------|----------|
| 위치 정보 | 러닝 경로 기록, 거리·페이스 계산, 영토 점령 기능 |
| 건강 정보 | 심박 기반 운동 강도 표시, Apple 건강 앱 동기화 |
| 모션 센서 | 케이던스·걸음수 측정 |
| 자동 생성 정보 | 사용자별 러닝 기록·통계·배지·챌린지 제공 |

수집한 정보는 **위 목적 이외의 용도로 이용되지 않습니다**.

---

## 3. 보관 및 파기

### 3-1. 보관 위치
- 모든 데이터는 **사용자 단말기 내부 저장소(AsyncStorage)** 에 보관됩니다.
- 별도의 서버로 전송하지 않으며, 클라우드에 저장하지 않습니다.
- Apple HealthKit 연동 시, 사용자가 동의한 항목에 한해 Apple 건강 앱에도 저장됩니다(Apple의 개인정보 처리방침 적용).

### 3-2. 보관 기간
- 사용자가 앱을 삭제하거나 앱 내 "모든 데이터 삭제" 메뉴를 실행하면 **즉시 파기**됩니다.
- 별도의 보관 기간은 없으며, 사용자가 직접 관리합니다.

### 3-3. 파기 방법
- 단말기 저장소에서 즉시 영구 삭제됩니다.

## 4. 제3자 제공 및 위탁

Notle Run은 사용자의 개인정보를 **제3자에게 제공하거나 위탁 처리하지 않습니다**.

다만 다음의 경우는 예외적으로 사용자의 단말기 내부에서 다른 시스템으로 데이터가 이동할 수 있으며, 이는 사용자의 명시적 동의에 따릅니다.

| 대상 | 항목 | 동의 시점 |
|------|------|----------|
| Apple HealthKit | 심박수, 걸음수, 거리, 워크아웃 | 첫 실행 시 사용자 동의 |
| 운영체제 위치 서비스 | GPS 좌표 | 위치 권한 허용 시 |

위 모든 이동은 사용자 단말기 내부에서 처리되며, 외부 서버로 전송되지 않습니다.

---

## 5. 광고 식별자 및 추적

Notle Run은 **광고 식별자(IDFA, AAID)를 수집·이용하지 않으며, 광고 네트워크에 데이터를 전송하지 않습니다.**

---

## 6. 사용자의 권리

사용자는 언제든지 다음 권리를 행사할 수 있습니다.

| 권리 | 행사 방법 |
|------|----------|
| 데이터 열람 | 앱 내 기록·통계 화면을 통해 직접 확인 |
| 데이터 정정 | 닉네임·설정 항목은 앱 설정에서 변경 |
| 데이터 삭제 | "설정 → 데이터 → 모든 데이터 삭제" 또는 앱 삭제 |
| 데이터 내보내기 | "설정 → 데이터 → 데이터 내보내기" 로 JSON 백업 |
| 권한 철회 | iOS 설정 → Notle Run → 권한 항목별 토글 |
